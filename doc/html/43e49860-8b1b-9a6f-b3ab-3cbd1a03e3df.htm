BufferedClient Events

The BufferedClient type exposes the following members.

Events
  NameDescription
OnConnected
Triggered when the client successfully connects to an OTEX server.
OnDisconnected
Triggered when the client is disconnected from an OTEX server. Boolean parameter is true if the disconnection was forced by the server.
OnDocumentChanged
Triggered when the buffered client's internal text has been changed. Bool parameter is true if the change was from a remote client.
OnMetadataUpdated
Triggered when a remote client updates it's metadata. Do not call any of this object's methods from this callback or you may deadlock!
OnThreadException
Triggered when an internal thread throws an exception.
(Inherited from ThreadController.)
