Diff Class
A diff generator based on Meyer's "An O(ND) Difference Algorithm and its Variations", 1986. This implementation is a generic, stripped-down and optimized version of Matthias Hertel's, found here: http://www.mathertel.de/diff/. The inline comments are from his original.
