The IClient type exposes the following members.

Events
  NameDescription
OnConnected
Triggered when the client successfully connects to an OTEX server.
OnDisconnected
Triggered when the client receives remote operations from the server. Do not call any of this object's methods from this callback or you may deadlock!
OnMetadataUpdated
Triggered when a remote client updates it's metadata. Do not call any of this object's methods from this callback or you may deadlock!
