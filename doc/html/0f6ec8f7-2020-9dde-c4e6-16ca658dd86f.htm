The ServerStartParams type exposes the following members.

Fields
  NameDescription
EditMode
How to handle the file given by Path if it already exists. True: reads the file in and uses it's contents as the starting point for the session document; False: session document is empty, and the existing file is overwritten when the server syncs to disk.
FilePath
Path to the text file you'd like clients to edit/create.
MaxClients
How many clients are allowed to be connected at once? Setting it to 0 means "no limit" (there is an internal maximum limit of 100)
Name
The friendly name of the server.
Password
A password required for clients to connect to this session (null == no password required).
Port
Listening for new client connections will bind to this port (supports IPv4 and IPv6).
Public
Advertise the presence of this server so it shows up in local server browsers.
ReplaceTabsWithSpaces
Replace tabs in loaded files with spaces? Useful if paired with a client editor which does not support \t characters (e.g. FastColoredTextBox).
